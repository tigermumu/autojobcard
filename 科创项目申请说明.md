# 飞机工卡智能匹配与处理系统
## 科创项目申请说明文档

---

## 一、项目概述

### 1.1 项目背景

在航空维修领域，工卡（Work Card）是飞机维修和检查任务的核心记录文件。传统的工卡处理流程存在以下痛点：

- **数据清洗效率低**：历史工卡数据格式不统一、字段不规范，需要大量人工清洗和分类
- **匹配准确率低**：缺陷清单与历史工卡的匹配依赖人工经验，准确率有限且耗时
- **知识传承困难**：维修经验难以系统化沉淀，新员工上手周期长
- **处理效率低**：单次缺陷清单处理需要数小时，影响维修进度

### 1.2 项目目标

本项目旨在构建一个**智能化的飞机工卡处理系统**，通过引入**大语言模型（LLM）技术**和**智能匹配算法**，实现：

1. **自动化数据清洗**：利用AI技术对历史工卡数据进行高效清洗、标准化和分类
2. **智能工卡匹配**：基于多维度相似度算法，自动匹配缺陷清单与历史工卡，推荐候选方案
3. **知识沉淀**：构建标准化的工卡知识库，支持多构型、多机型的统一管理
4. **效率提升**：将缺陷清单处理时间从数小时缩短至分钟级，准确率提升至85%以上

### 1.3 应用场景

- **飞机维修方案制定**：快速匹配历史维修方案，辅助制定新维修计划
- **缺陷处理决策支持**：为飞机缺陷清单自动推荐最匹配的工卡方案
- **历史数据挖掘**：清洗和整理历史工卡数据，构建可检索的知识库
- **多构型管理**：支持A320、A321、B737等多种机型构型的统一管理

## 三、核心功能模块

### 3.1 模块一：历史工卡数据清洗与存储

#### 功能描述
将原始的、未分类的历史工卡数据转化为结构化、高质量的、按构型分类的工卡基础数据库。

#### 核心能力

**1. 构型索引文件管理**
- 支持多构型（A320、A321、B737等）的索引文件配置
- 树形编辑器可视化编辑索引结构
- 9字段标准化索引数据表：
  - 4个层级字段：主区域 → 主部件 → 一级子部件 → 二级子部件
  - 5个独立对照字段：方位、缺陷主体、缺陷描述、位置、数量
- 支持索引数据的导入、导出和版本管理

**2. AI辅助数据清洗**
- **字段标准化**：依据索引文件，将原始工卡数据字段映射至标准字段
- **AI对比校验**：使用大语言模型对工卡数据的关键描述字段进行语义校验
  - 识别异常数据点
  - 确保同构型下数据一致性
  - 自动标记潜在问题
- **批量处理**：支持大规模历史数据的批量清洗
- **清洗日志**：记录清洗过程，支持追溯和审计

**3. 工卡数据分类存储**
- 按构型严格分类存储
- 支持单机构型识别（飞机号、MSN、AMM/IPC EFF）
- 工卡类型分类（例行卡、非例行卡等）
- 支持工卡数据的查询、筛选和导出

#### 技术亮点
- **LLM语义理解**：利用大语言模型理解工卡描述的语义，而非简单的关键词匹配
- **多维度标准化**：通过9字段索引数据实现多维度数据标准化
- **可配置清洗规则**：支持不同构型的自定义清洗规则

### 3.2 模块二：索引数据管理

#### 功能描述
构建标准化的索引数据体系，为数据清洗和匹配提供基准。

#### 核心能力

**1. 层级索引数据管理**
- 树形结构可视化编辑
- 扁平化存储，支持快速查询
- 支持批量导入和导出
- 数据验证和完整性检查

**2. 权重配置**
- 可配置的字段权重参数
- 二级子部件权重最高（35%），用于精确匹配
- 支持动态调整权重，优化匹配准确率

**3. 多构型支持**
- 每个构型独立的索引数据表
- 支持构型间的数据复制和迁移
- 构型版本管理

#### 技术亮点
- **层级权重算法**：通过权重配置实现精准匹配
- **树形数据转换**：高效的扁平化存储和树形展示转换

### 3.3 模块三：缺陷清单处理与工卡匹配

#### 功能描述
处理用户提交的缺陷清单，进行智能匹配计算，并提供人机交互界面进行最终确认和录入。

#### 核心能力

**1. 缺陷清单提交与清洗**
- Excel/CSV文件上传
- 自动识别缺陷清单格式
- AI辅助清洗和标准化
- 按飞机号分类存储

**2. 智能匹配算法**
- **多维度相似度计算**：
  - 基于索引数据的层级匹配（主区域→主部件→一级子部件→二级子部件）
  - TF-IDF文本相似度计算
  - 余弦相似度向量化匹配
- **权重加权求和**：
  - 二级子部件：35%（最高权重）
  - 一级子部件：20%
  - 主部件：15%
  - 主区域：10%
  - 其他字段：20%
- **筛选优化**：
  - 第一级筛选：根据关键信息（系统、部件、位置）快速过滤
  - 第二级匹配：对筛选结果进行精确相似度计算
- **相似度评分**：输出0-100分的相似度评分

**3. 候选工卡推荐**
- 自动筛选相似度≥75分的工卡
- 按相似度降序排列
- 显示匹配详情和评分依据
- 支持批量匹配和进度跟踪

**4. 人机交互确认**
- 缺陷记录与候选工卡并排展示
- 用户单项选择最匹配工卡
- 实时显示匹配进度
- 支持跳过和标记

**5. 工卡录入集成**
- 自动整合缺陷清单和选定工卡数据
- 通过VPN/API向公司工卡系统执行录入
- 支持批量录入和错误重试
- 录入日志和状态跟踪

**6. 落选清单管理**
- 清晰展示未匹配或低分匹配的缺陷记录
- 支持导出为Excel/CSV格式
- 支持人工复核和二次处理

#### 技术亮点
- **混合匹配算法**：结合规则匹配和语义匹配，提升准确率
- **异步处理**：后台任务处理，不阻塞用户操作
- **进度跟踪**：Redis实时更新匹配进度
- **可配置阈值**：支持调整相似度阈值，适应不同场景

### 3.4 模块四：批量工卡导入

#### 功能描述
从公司工卡系统批量导入工卡数据，支持历史工卡和当前工卡的导入。

#### 核心能力
- 批量导入：支持一次导入多个工卡
- 数据转换：自动转换公司系统格式到标准格式
- 导入日志：记录导入过程和结果
- HTML快照：保存导入时的页面快照，便于追溯
